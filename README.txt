# 🦴 骨质疏松机会性筛查系统

基于SVM机器学习模型的骨质疏松风险预测系统，使用腰椎CT值进行机会性筛查。

## 📊 模型性能

| 指标 | 数值 |
|------|------|
| 准确率 | 83.05% |
| AUC | 0.9074 |
| 敏感性 | 90.62% |
| 特异性 | 74.07% |
